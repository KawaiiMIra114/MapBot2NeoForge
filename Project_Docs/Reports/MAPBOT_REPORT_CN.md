# MapBot Reforged v5.0.0 技术报告

## 摘要

MapBot Reforged 是一款 NeoForge 模组，通过 OneBot 协议连接 Minecraft 服务器与 QQ 群。v5.0.0 版本新增了增强的服务器管理功能、性能监控集成和用户体验改进。

## 架构概览

```
┌─────────────────┐    WebSocket     ┌─────────────────┐
│    QQ 客户端     │◄───────────────►│   OneBot API    │
│   (手机/电脑)    │                  │  (go-cqhttp)    │
└─────────────────┘                  └────────┬────────┘
                                              │
                                              ▼
┌─────────────────┐     事件         ┌─────────────────┐
│   Minecraft     │◄───────────────►│    MapBot       │
│     服务器       │                  │   Reforged      │
└─────────────────┘                  └─────────────────┘
```

## 新功能 (v5.0.0)

### 1. Duolingo Super 集成
| 命令 | 说明 |
|------|------|
| `#report` | 读取 `duolingo_super_report.json` 并发送格式化的性能摘要到 QQ 群 |

**报告包含：**
- 总压缩统计
- LZ4/Zstd 数据包数量
- 节省字节计算
- 过滤器统计

### 2. 服务器关闭管理
| 命令 | 说明 |
|------|------|
| `#stopserver [秒数]` | 启动倒计时关服（默认60秒） |
| `#cancelstop` | 取消正在进行的关服（仅管理员） |

**功能特点：**
- QQ 群和游戏内同步倒计时公告
- 支持优雅取消
- 最少10秒倒计时

### 3. 命令冷却系统
- 每用户5秒命令冷却
- 管理员豁免冷却
- 防止命令刷屏

### 4. 服务器生命周期消息
| 事件 | 消息 |
|------|------|
| 服务器启动 | "早安! 服务器已启动"（延迟5秒发送） |
| 服务器关闭 | "晚安! 服务器即将关闭" |

## 命令参考

### 玩家命令
| 命令 | 别名 | 说明 |
|------|------|------|
| `#id <名字>` | `#bind`, `#绑定` | 绑定 QQ 到游戏账号 |
| `#unbind` | `#解绑` | 解绑账号 |
| `#list` | `#在线` | 显示在线玩家 |
| `#status` | `#tps`, `#状态` | 服务器状态 |
| `#playtime` | `#在线时长` | 查询游戏时长 |
| `#report` | `#报告` | 性能报告 |
| `#help` | `#菜单` | 显示帮助 |

### 管理命令（仅管理群）
| 命令 | 说明 |
|------|------|
| `#inv <名字> [-e]` | 查看背包/末影箱 |
| `#location <名字>` | 查询玩家位置 |
| `#stopserver [秒]` | 关闭服务器 |
| `#cancelstop` | 取消关服 |
| `#addadmin <QQ>` | 添加管理员 |
| `#removeadmin <QQ>` | 移除管理员 |
| `#adminunbind <名字>` | 强制解绑玩家 |
| `#reload` | 重载配置 |

## 配置文件

**路径：** `config/mapbot-common.toml`

```toml
[bot]
websocketUrl = "ws://127.0.0.1:8080"
playerGroupId = 123456789
adminGroupId = 987654321
reconnectDelay = 5000

[features]
enableCooldown = true
cooldownSeconds = 5
```

## 技术实现

### WebSocket 处理器
- 自动重连（可配置延迟）
- 离线消息队列缓冲
- 心跳监控

### 双群支持
- 玩家群：通用命令、消息转发
- 管理群：管理命令、警报通知

### CQ 码解析器
- 提取 @提及 用于定向通知
- 回复消息上下文处理

- NeoForge 21.1+
- Minecraft 1.21.1
- OneBot v11 兼容机器人（推荐 go-cqhttp）

## 已知问题

1. WebSocket 重连可能需要最多30秒
2. 大型背包查询在慢速连接上可能超时
