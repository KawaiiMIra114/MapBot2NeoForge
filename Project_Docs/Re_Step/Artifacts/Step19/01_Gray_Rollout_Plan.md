### 阶段 0: 暗发布 (Dark Launch)
| 项目 | 值 |

| 目标 | 验证部署完整性、配置加载、Bridge 连接 |

| 责任人 | Solo Maintainer |

## 不可跳级规则
1. **强制顺序**: 暗发布 → 5% → 25% → 100%，不可越级。
2. **无快速通道**: 无论指标多好，每阶段必须完成最短观察时间。
3. **回滚只退一级**: 超阈值回滚到前一阶段，不跳级回退。
4. **二次进入**: 回滚后重新进入该阶段需新一轮完整观察。
