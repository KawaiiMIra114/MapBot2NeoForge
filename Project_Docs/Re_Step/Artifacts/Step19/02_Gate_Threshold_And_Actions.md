## 自动回滚动作

### 触发流程
```
检测超阈值 → 确认持续3min → 锁定变更 → 回滚到前一阶段 → 验证恢复 → 记录RCA
```

### 回滚时效
| 步骤 | 目标时间 |
|---|---|
| 检测到触发 | ≤1min |
| 回滚启动 | ≤2min |
| 回滚完成 | ≤5min |
| 恢复验证 | ≤10min |

### 回滚后验证
1. 指标回到基线 ±0.2% 区间 (10 分钟内)
2. 核心冒烟测试全部通过
3. Bridge 重连成功 + 心跳正常
4. 状态可追踪 (日志 + 指标 + 告警记录)
