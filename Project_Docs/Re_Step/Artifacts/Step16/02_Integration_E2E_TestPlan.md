## 通过标准
1. 关键链路 (绑定/解绑/跨服/重载/命令) 四类场景覆盖率 100%
2. 每次失败可定位到链路节点 (Alpha/Bridge/Reforged)
3. requestId 全链路可追踪

## 失败判据
1. 仅有成功路径测试
2. 失败场景无可诊断断言
3. 超时/重试场景缺失
